# KUBECTL

Pythonic version of Kubernetes API, showcases the use of AioFauna library  while being inspired on Kubernetes API (KubeCTL CLI) created by Google on 2015. It comes with a GUI instead.

## Next Features

* OAuth2 con Github
* Webhook desde Github
